# Campus Green Habit Tracker

## Project Idea

This project is a simple web application that helps students track eco-friendly habits on campus. The goal is to encourage small daily actions that contribute to sustainability, such as recycling and reducing energy usage.

Users can check completed habits and earn points. The total score is displayed on the page.

---

## Features

- Checklist of eco-friendly campus habits  
- Automatic score calculation  
- Weekly reset button  
- Progress saved using localStorage  
- Simple and clean user interface  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  

No external libraries were used.

---

## How to Run the Project

1. Download the project files  
2. Open the folder  
3. Double-click on `index.html`  
4. The application will open in your browser  

No installation or server setup is required.

---

## How to Use

- Check the habits you completed  
- The total points will update automatically  
- Your progress is saved even if you refresh the page  
- Click **Reset Week** to clear all progress and start again
